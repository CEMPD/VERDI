Detailed Description

A modified version of Quantize.java that handles alpha. An efficient color quantization algorithm, adapted from the C++ implementation quantize.c in ImageMagick. The pixels for an image are placed into an oct tree. The oct tree is reduced in size, and the pixels from the original image are reassigned to the nodes in the reduced tree.
