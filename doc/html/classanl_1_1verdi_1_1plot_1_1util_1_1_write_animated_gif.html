Detailed Description

Creates an animated GIF from GIF frames. A thin wrapper to code written by other people, as documented on the thread on the Sun forums 'Create animated GIF using imageio' http://forums.sun.com/thread.jspa?threadID=5395006 See the printUsage() method for details on parameters required.
