Detailed Description

This is a layout that strips from the messages the internal anchor of the message center. So a message goes from "[main] FATAL MessageCenter.INTERNAL.simphony.util.messages.Log4jMessageListener blas" to "[main] FATAL simphony.util.messages.Log4jMessageListener blas"
