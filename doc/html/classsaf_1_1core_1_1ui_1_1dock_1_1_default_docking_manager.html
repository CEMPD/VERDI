DockableFrame saf.core.ui.dock.DefaultDockingManager.createDockable ( String  id,
JComponent  comp,
MinimizeLocation  location,
int  dockableControls 
)

Creates a dockable frame with the specified id, hide location, window controls and containing the specified component. This only creates the dockable frame. To add a created dockable frame into a window use the addDockableToGroup method.

The dockable frame controls are:

  • DockingManager.CLOSE
  • DockingManager.HIDE
  • DockingManager.MAXIMIZE
  • DockingManager.FLOAT

These can be logically or'ed together.

Parameters
idthe id that uniquely identifies the created Dockable frame
compthe component displayed by the dockable frame
locationthe location where the dockable frame is minimized to
dockableControlsspecifies what controls (close, maximize, float etc.) are displayed on the dockable frame's title bar
Returns
the created dockable frame
