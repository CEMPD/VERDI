Detailed Description

This handles IO operations on simphony.settings.SettingsRegistrys. It does not handle converting between different implementations of simphony.settings.SettingsRegistrys, so you cannot serialize one implementation and read it in with a different implementation.

static SettingsRegistry simphony.settings.SettingsIO.loadSettings ( String  registryId,
String  fileName 
)
static

This loads from the specified file a SettingsRegistry. If the file does not exist this will return a new SettingsRegistry with the given id, otherwise it will load one from the specified file and set its id to be the specified id.

If the registry has already been loaded this will return the previously loaded one. As with the store methods, this will not attempt to reload the next registry in the chain.

Parameters
registryIdthe id for the registry
fileNamethe name of the file that the registry may be loaded from
