org.libtiff.jai.codecimpl.XTIFFImage.XTIFFImage ( SeekableStream  stream,
TIFFDecodeParam  param,
int  directory 
) throws IOException

Constructs a XTIFFImage that acquires its data from a given SeekableStream and reads from a particular IFD of the stream. The index of the first IFD is 0.

Parameters
streamthe SeekableStream to read from.
paraman instance of TIFFDecodeParam, or null.
directorythe index of the IFD to read from.
