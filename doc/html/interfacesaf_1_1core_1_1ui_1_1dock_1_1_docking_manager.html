DockableFrame saf.core.ui.dock.DockingManager.createDockable ( String  id,
JComponent  comp,
MinimizeLocation  location,
int  dockableControls 
)

Creates a dockable with the specified id, hide location, window controls and containing the specified component. This only creates the dockable. To add a created dockable into a window use the addDockableToGroup method.

The dockable controls are:

  • DockingManager.CLOSE
  • DockingManager.HIDE
  • DockingManager.MAXIMIZE
  • DockingManager.FLOAT

These can be logically or'ed together.

Parameters
idthe id that uniquely identifies the created Dockable
compthe component displayed by the dockable
locationthe location where the dockable is minimized to
dockableControlsspecifies what controls (close, maximize, float etc.) are displayed on the dockable's title bar
Returns
the created dockable
