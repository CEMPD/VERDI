Detailed Description

File Name:SwingWorker.java Description: Code to invoke a task when under swing Based on code off Sun site

Version
May 2004
Author
Mary Ann Bitz
Argonne National Lab This is the 3rd version of SwingWorker (also known as SwingWorker 3), an abstract class that you subclass to perform GUI-related work in a dedicated thread. For instructions on and examples of using this class, see:

http://java.sun.com/docs/books/tutorial/uiswing/misc/threads.html

Note that the API changed slightly in the 3rd version: You must now invoke start() on the SwingWorker after creating it.
