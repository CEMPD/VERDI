Known Bugs
==========

You are encouraged to review, review the latest release notes: https://github.com/CEMPD/VERDI/blob/master/doc/Release_Notes/README.md query the CMAS Forum <https://forum.cmascenter.org/c/verdi/20> to search questions and answers, bug reports, and suggestions. Once a bug is identified through the CMAS User Forum, it will be added as an issue on the Github for the developers to prioritize and fix. https://github.com/CEMPD/VERDI/issues.
