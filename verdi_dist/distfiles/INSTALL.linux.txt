1. tar -xzvf verdi_1.3.tar.gz into the appropriate place.

2. Java is provided in verdi_1.3.tar.gz file for 32 bit linux. 
If you would like to install 64 bit linux follow step 3.  Otherwise skip to step 6.

3. download Java for your platform if it isn't already installed
   for linux:  http://www.java.com/en/download/manual.jsp

4. Download Java3D for your platform if it isn't already installed
For installation instruction please go to:  
	http://www.oracle.com/technetwork/java/javase/tech/index-jsp-138252.html

5. Download JAI 1.1.3 for your platform if it isn't already installed
For installation instruction please go to:
	http://java.sun.com/products/java-media/jai/downloads/download-1_1_3.html

6. Edit verdi_1.3/verdi.sh
change path for VERDI_HOME variable to reflect where verdi has been installed,
change path for JAVA variable to reflect where the java executable has been installed

7. VERDI can be run using the verdi.sh executable script.
